# calculus-engine
Advanced Calculus Solver with step-by-step solutions (Derivatives, Integrals, Double &amp; Triple). Primarily for learnign approach.
## Advanced Calculus Solver — MathWithAmrit
A beautiful, step-by-step calculus solver built with Streamlit and SymPy.

### Features

- **Derivative Solver**  
  - Power, product, quotient, and chain rules  
  - Up to 5th order derivatives  
  - Tangent line computation  
  - Critical and inflection points  

- **Single Integral**  
  - Definite and indefinite integrals  
  - u-substitution and integration by parts  
  - Fundamental Theorem of Calculus (FToC)  
  - Riemann sums  

- **Double Integral**  
  - Fubini's theorem  
  - 3D surface visualization  
  - Contour plot explorer  

- **Triple Integral**  
  - Iterated 3D integration  
  - Z/X/Y slice heatmaps  

- **User Experience**  
  - Dark / Light theme  
  - Mobile-friendly interface  
  - Formula reference  
  - Interactive graphs
